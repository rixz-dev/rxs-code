Cara penggunaan:
- git clone https://github.com/rixz-dev/rxs-code.git
- cd rxs-code
- chmod +x bin/rxs-code
- npm install
- pkg install fzf which
- cp .env.example .env
- nano .env #isi api key yang dibutuhkan lalu ctrl +x y
- rxs-code

Command :
- /provider : pilih provider
- /help : lihat semua bantuan
- /model : jika ingin model costum, masukkan model id, misalnya anda ingin model dari Nvidia, ketik /provider nvidia terlebih dahulu, lalu ketik /model minimax/minimax-m2.7 selesai.
- Saran model terbaik dan optimal (openrouter) id : deepseek/deepseek-v4-flash
